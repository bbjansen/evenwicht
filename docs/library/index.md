# Evenwicht Library

Evenwicht is a typed mathematics library for TypeScript. It covers
algebraic structures, mathematical operators, probability and statistics.
Every public function maps to a chapter in the textbook; the chapter
provides the theory, the library provides the computation.

> The library is under active development. The 48 chapters of theory and
> the [API reference](../api/01-expressions.md) define the target
> interface. This section will be updated as modules are completed.

## Installation

```bash
npm install evenwicht
```

Requires Node.js >= 20.0.0. The package ships ES modules with full
TypeScript type declarations.
