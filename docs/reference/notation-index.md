## Greek Letters

- **$\alpha$**: Significance level (probability of Type I error); Order of the fractional derivative/integral (real number); Thermal diffusivity $\alpha = k/(\rho c_p)$ ($\text{m}^2 \text{s}^{-1}$); Planetary albedo (dimensionless, $0 \leq \alpha \leq 1$); current value $\alpha \approx 0.30$; Substitution rate parameter (Jukes–Cantor model); [16](../domains/16-statistical-inference.md), [24](../domains/24-fractional-calculus.md), [32](../domains/32-energy-systems.md), [38](../domains/38-climate-modeling.md), [48](../domains/48-genetics.md)
- **$\alpha_i$**: Jensen's alpha: excess return unexplained by factor exposure; [27](../domains/27-quantitative-trading.md)
- **$B(\nu, T)$**: Planck blackbody spectral radiance ($\text{W sr}^{-1} \text{m}^{-2} \text{Hz}^{-1}$); [46](../domains/46-cosmology.md)
- **$\beta$**: Probability of Type II error; Transmission rate (contacts per unit time $\times$ probability of transmission per contact); [16](../domains/16-statistical-inference.md), [30](../domains/30-epidemiology.md)
- **$\beta_i$**: CAPM beta of asset $i$: $\text{Cov}(R_i, R_m)/\text{Var}(R_m)$; [27](../domains/27-quantitative-trading.md)
- **$\boldsymbol{\beta}$**: The $(p+1) \times 1$ vector of true regression coefficients $(\beta_0, \beta_1, \ldots, \beta_p)'$; [17](../domains/17-regression.md)
- **$\boldsymbol{\lambda}$**: Vector of Lagrange multipliers $(\lambda_1, \ldots, \lambda_m)$; [12](../domains/12-constrained-optimization.md)
- **$\boldsymbol{\mu}$**: Vector of expected returns $(\mu_1, \ldots, \mu_n)'$; [27](../domains/27-quantitative-trading.md)
- **$\boldsymbol{\omega}$**: Angular velocity vector for plate rotation (rad/Myr); [45](../domains/45-geology-seismology.md)
- **$\boldsymbol{\tau}$**: Applied joint torque vector; [42](../domains/42-robotics.md)
- **$\boldsymbol{\theta}$**: Vector of joint angles $(\theta_1, \ldots, \theta_n)^T$; [42](../domains/42-robotics.md)
- **$\boldsymbol{\varepsilon}$**: The $n \times 1$ error vector; [17](../domains/17-regression.md)
- **$c, \alpha, \beta$**: Scalars (real numbers); [8](../domains/08-vectors.md)
- **$\chi^2$**: Chi-square test statistic; [48](../domains/48-genetics.md)

- **$\delta$**: A positive real number representing a tolerance on the input side of a limit; Discount factor: $\delta = 1/(1+r)$; Depreciation rate; [3](../domains/03-limits-continuity.md), [25](../domains/25-financial-mathematics.md), [33](../domains/33-equilibrium.md)
